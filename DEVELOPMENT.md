## How to build Astate

Please checkout the source tree from https://github.com/inclusionAI/asystem-astate.

### Build

### Build Astate

```bash
# install deps
make deps

# build as develop and install astate into python env
make develop

# build release and install
make install
```

#### Environment Requirements

- gcc >= 10 or clang >= 15
- python >= 3.10
